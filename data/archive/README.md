# Data Archive

Completed dispatches, processed completion reports, closed story specs, and historical work records live here.

Use this folder to keep active workflow directories small:

- `data/dispatches/` should contain only active dispatches.
- `data/completions/` should contain only unprocessed completion reports.
- `data/escalations/` should contain only unresolved escalations.
- Durable learning should be summarized into `data/wiki/`.

Do not archive secrets or client-sensitive material. Redact or remove sensitive details before committing.
